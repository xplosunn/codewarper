Codewarper

The coding agent you can standardize on.

No magic. Under your control.

You can add the config file to your repo.

No tools by default

Out of the box it can't touch your files, run shell commands, or make network calls. You add the tools you want.

No telemetry

It doesn't phone home. At all. The only outbound connections are the ones you set up yourself, to the provider APIs.

Auditable source code

Source is on GitHub. You can know exactly what you're running.

Zero dependencies

No risk of supply-chain attacks. The only thing you're trusting is Node.js.

Get started

Install it.

npm install -g codewarper

Run it in any project.

codewarper
npm GitHub
